# 认知工程理论

认知工程（Cognitive Engineering）是多学科交叉领域，研究如何将认知科学原理应用于人机系统的设计、分析与评估。以下收集中核理论分支。

## 目录

| 理论分支 | 核心关注 | 关键人物 | 文件 |
|---------|---------|---------|------|
| 心智模型 (Mental Models) | 用户脑中如何表征系统结构 | Craik, Johnson-Laird, Senge | [mental-models.md](mental-models.md) |
| 图式理论 (Schema Theory) | 认知基本单元的结构与演化 | Bartlett, Piaget, Rumelhart | [schema-theory.md](schema-theory.md) |
| 分布式认知 (Distributed Cognition) | 认知跨越人、工具、环境的分布 | Hutchins | [distributed-cognition.md](distributed-cognition.md) |
| 认知任务分析 (Cognitive Task Analysis) | 任务中认知需求的系统分析方法 | Kieras, Schraagen | [cognitive-task-analysis.md](cognitive-task-analysis.md) |
| 情境认知 (Situated Cognition) | 认知嵌入具体情境而非抽象 | Suchman, Lave | [situated-cognition.md](situated-cognition.md) |
| 生态界面设计 (Ecological Interface Design) | 界面应直接呈现系统深层结构 | Vicente, Rasmussen | [ecological-interface-design.md](ecological-interface-design.md) |
