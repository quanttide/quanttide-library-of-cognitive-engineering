# 图式理论 (Schema Theory)

## 概述

图式是认知的基本单元——一种用于组织和解释信息的心理结构。图式理论解释人如何用已有知识框架来处理新信息、填补信息空白、做出推断。

## 起源

- **Frederic Bartlett (1932)** — 《Remembering》通过"幽灵之战"实验发现记忆是被图式重构而非照实复述
- **Jean Piaget (1950s)** — 认知发展理论：同化（新信息融入已有图式）vs 顺应（改变图式以适应新信息）
- **Rumelhart & Norman (1970s)** — 将图式引入信息加工心理学
- **Minsky (1975)** — 提出"框架"（Frame）概念，接近图式的 AI 等价物

## 核心特征

- **知识封装**：图式将经验抽象为通用结构，省略具体细节
- **层次组织**：主图式包含子图式，形成嵌套结构
- **填充默认值**：信息缺失时，图式提供"默认值"填补认知缺口
- **主动选择**：图式决定哪些信息被注意、如何被解释
- **难以改变**：顽固图式（特别是社会偏见）会抵抗矛盾证据

## 图式的生命周期

1. **形成**：反复暴露于同类情境，逐渐抽象出共性
2. **强化**：一致的经验使图式更稳固、更易激活
3. **调整**：有限的不一致导致图式边缘调整
4. **重构**：大量矛盾证据迫使图式整体重组（Paradigm shift）

## 在认知工程中的应用

| 应用 | 说明 |
|------|------|
| 界面设计 | 匹配用户已有的图式降低学习成本（如购物车图标） |
| 心智模型建模 | 将用户图式显式化为可分析的结构（本仓库 context/schema.md） |
| 错误预测 | 知道哪些图式容易导致特定错误路径 |
| 渐进式引导 | 不强行纠正错误图式，而是用逐步反例促成重构 |
